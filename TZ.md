# Техническое задание: BookMood - Сервис подбора книг по настроению

## 1. Цель проекта
Разработка веб-сервиса для персонализированного подбора книг на основе текущего эмоционального состояния пользователя, уровня его загруженности и литературных предпочтений.

## 2. Роли пользователей
- **Гость:** Просмотр каталога книг, использование формы подбора, просмотр статистики
- **Зарегистрированный пользователь:** Все функции гостя + сохранение истории подборок, избранные книги
- **Администратор:** Полный доступ к управлению контентом через админ-панель Django

## 3. Модели данных
### Модель 1: Book (Книга)
- `title` (CharField) - Название книги
- `author` (CharField) - Автор
- `mood` (CharField) - Настроение книги (8 категорий)
- `complexity` (CharField) - Уровень сложности (3 уровня)
- `description` (TextField) - Описание

### Модель 2: UserProfile (Профиль пользователя)
- `user` (OneToOneField) - Связь с пользователем Django
- `favorite_genres` (CharField) - Любимые жанры
- `reading_speed` (CharField) - Скорость чтения
- `favorite_books` (ManyToManyField) - Избранные книги

### Модель 3: BookSelection (История подборок)
- `user` (ForeignKey) - Пользователь
- `selected_mood` (CharField) - Выбранное настроение
- `selected_complexity` (CharField) - Выбранная сложность
- `selected_date` (DateTimeField) - Дата подбора
- `recommended_books` (ManyToManyField) - Рекомендованные книги

## 4. Ключевой функционал (User Stories)
1. **Просмотр каталога:** Пользователь может просматривать все книги с фильтрацией
2. **Подбор по настроению:** Заполнение формы с параметрами и получение рекомендаций
3. **Статистика:** Визуализация данных о книгах с помощью Chart.js
4. **Поиск:** Поиск книг по названию и автору
5. **Администрирование:** Управление контентом через встроенную админку Django

## 5. Технический стек и интеграции
- **Backend:** Django 4.2, Python 3.10
- **Frontend:** Bootstrap 5, Chart.js для графиков
- **База данных:** SQLite (разработка)
- **Аналитика:** Django ORM агрегация, визуализация Chart.js
- **Формы:** Django Forms с валидацией
